# Samsung-Sales-Analysis
# Samsung Sales Analysis Dashboard using Power BI

This project is designed to analyze Samsung sales data and transform raw data into meaningful business insights using Power BI. The dashboard helps in monitoring overall sales performance, identifying top-selling products, tracking revenue growth, and understanding customer purchasing behavior through interactive visualizations.

The main purpose of this project is to help businesses make data-driven decisions by analyzing sales trends, regional performance, and product demand in an efficient and visually understandable way.

# Project Overview

In this project, sales data was collected from Excel/CSV files and processed in Power BI. The dataset includes information such as product names, sales amount, quantity sold, customer details, regions, dates, revenue, and profit.

After importing the dataset, the data was cleaned and transformed using Power Query. Missing values, duplicate records, and incorrect data types were handled to prepare the data for analysis.

The Power BI dashboard contains multiple interactive reports and visualizations that provide detailed insights into business performance.
